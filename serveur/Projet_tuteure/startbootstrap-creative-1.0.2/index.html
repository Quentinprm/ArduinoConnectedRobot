Description de notre projet tutoré :

L’objectif du projet est la réalisation de robot piloté par internet via un navigateur (ou, à défaut, via un mobile/tablette). Basé sur châssis de grande série, équipé de divers capteurs (encodeurs, capteur directionnel de distance…), d’un microcontrôleur type Arduino ou mini PC type Raspberry Pi, d’une liaison sans fil wifi (ou autre avec base émetteur-récepteur fixe), relié à internet (ou un réseau local) via Wifi ou Ethernet depuis le robot ou depuis sa base, chaque robot aura son propre serveur embarqué ou distant permettant de le piloter en direct ou de lui apprendre un parcours qu’il saura répéter. Idéalement ce seront 2 robots qui pourront être réalisés avec différents châssis et technologies afin de les confronter dans une sorte de mini concours. En application domotique, on imagine un robot qui se rend à la demande dans un endroit précis du bâtiment pour y réaliser une surveillance ou une activité. Le site web permettra d’envoyer des commandes de base (avance, tourne…), programmer un chemin (en mode apprentissage ou en indiquant les points de passage), surveiller l’état du robot, l’historique de ses déplacements… Ce projet s’appuiera sur le club étudiant bidouilleMaker disposant du matériel nécessaire et savoir-faire adéquate.
